==========================================================
   GHP MANAGEMENT SYSTEM - ADMIN FEATURES SUMMARY
   Date: 2026-03-06
==========================================================

1. DASHBOARD & NAVIGATION (ระบบนำทาง)
----------------------------------------------------------
- Admin Direct Login: แอดมินล็อกอินแล้วเข้าหน้า Admin Panel ทันที (Bypass Dashboard ทั่วไป)
- Sidebar Menu: เมนูทางซ้ายแบ่งสัดส่วนชัดเจน 
   - Admin Profile (จัดการตนเอง)
   - User Management (จัดการพนักงาน)
   - Role & Permission (จัดการสิทธิ์)
- Mobile Responsive: รองรับการใช้งานบนมือถือและแท็บเล็ต พร้อมเมนูสไลด์เปิด-ปิด

2. USER MANAGEMENT (การจัดการพนักงาน)
----------------------------------------------------------
- Employee List Table: ตารางรายชื่อพนักงานพร้อมข้อมูลสรุป
- Search & Filter: 
   - ค้นหาด้วย ชื่อ-นามสกุล / Username
   - กรองตามแผนก (Department)
- Sorting System: เรียงตาม แผนก หรือ วันที่เริ่มงาน (มากไปน้อย / น้อยไปมาก)
- Real-time Stats: แสดงจำนวนพนักงานทั้งหมด และแยกตามแผนกด้านบนตาราง
- Avatar View: กดดูรูปพนักงานขนาดใหญ่ได้ทันทีจากตาราง

3. EMPLOYMENT & ACCESS CONTROL (การตั้งค่าตำแหน่งและสิทธิ์)
----------------------------------------------------------
- Department Assign: เลือกฝ่ายงาน (Office, Draftman, Production, Installation, Management)
- Job Title Selection: เลือกตำแหน่งงาน (สัมพันธ์ตามแผนกที่เลือก)
- System Role: กำหนดระดับสิทธิ์ในระบบ (Admin, Manager, Employee, Inspector)
- Employment Status: สถานะการจ้างงาน (Intern, Permanent, Terminated)
- Auto-Termination: บันทึกวันที่ออกจากงานอัตโนมัติเมื่อเปลี่ยนสถานะเป็น Terminated

4. EMPLOYEE IDENTITY EDITOR (การจัดการข้อมูลส่วนตัวและเอกสาร)
----------------------------------------------------------
- Identity Form: แก้ไขข้อมูล ชื่อ, นามสกุล, ชื่อเล่น, วันเกิด, เบอร์โทร, สัญชาติ
- ID Document Management: จัดการเลขบัตรประชาชน / พาสปอร์ต
- Photo Upload Service: 
   - รองรับรูปถ่ายหน้าตรง (Portrait 3:4) พร้อม Preview
   - รองรับบัตรประชาชน/เอกสาร (Landscape 8:5) รองรับทั้งรูปภาพและ PDF
- Quick Switch Sidebar: แถบรายชื่อพนักงานด้านข้างสำหรับการสลับคนเพื่อแก้ไขข้อมูลได้อย่างรวดเร็ว

5. ROLES & PERMISSIONS (การจัดการสิทธิ์เข้าถึง)
----------------------------------------------------------
- Role Creation: สร้างตำแหน่งงานในระบบใหม่ได้
- Fine-grained Permissions: กำหนดสิทธิ์ย่อยได้ (เช่น สิทธิ์เข้าดูโปรเจกต์, สิทธิ์จัดการผู้ใช้)
- Security Bypass: แอดมินหลัก (Admin Role/Username) มีสิทธิ์ข้ามการเช็คย่อยเพื่อความรวดเร็ว

6. SECURITY & PROTECTION (ระบบความปลอดภัย)
----------------------------------------------------------
- Master Admin Locked: บัญชีแอดมินหลัก (admin) จะถูก "ล็อก" ไม่ให้ถูกแก้ไขหรือลบชื่อทิ้ง เพื่อป้องกันระบบล่ม
- 403 Forbidden Protection: ระบบหลังบ้านป้องกันการเข้าถึงจากผู้ที่ไม่มีสิทธิ์จัดการข้อมูลพนักงาน
- Logout System: ปุ่มออกจากระบบที่เข้าถึงได้ง่ายจากทุกหน้า
